# Dark Club - Баттл Манипуляций

Обучающая игра для изучения техник манипуляций и защиты от них.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Описание

Игра "Баттл Манипуляций" помогает:
- Изучить техники манипуляций
- Научиться распознавать манипуляции
- Освоить защитные техники (фокусы языка)
- Практиковаться в реальных сценариях

## Правила игры

1. **Фаза подготовки**: Манипулятор выбирает карты (роль, цель, технику, мишень)
2. **Фаза диалога**: Проводится диалог между манипулятором и жертвой
3. **Фаза раскрытия**: Жертва пытается угадать цель, технику и мишень
4. **Подсчет очков**: Очки начисляются за успешные манипуляции и правильные угадывания
